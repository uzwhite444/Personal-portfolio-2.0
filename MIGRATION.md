# 🔄 Руководство по миграции

## Как заменить старую версию на новую

### Вариант 1: Полная замена (рекомендуется)

1. **Создайте резервную копию**
\`\`\`bash
# Переименуйте старую папку
mv My-personal-portfolio My-personal-portfolio-old
\`\`\`

2. **Скопируйте новую версию**
\`\`\`bash
# Скопируйте всю папку portfolio-v2 в My-personal-portfolio
cp -r portfolio-v2 My-personal-portfolio
\`\`\`

3. **Установите зависимости**
\`\`\`bash
cd My-personal-portfolio
npm install
\`\`\`

4. **Запустите проект**
\`\`\`bash
npm start
\`\`\`

---

### Вариант 2: Постепенная миграция

Если хотите сохранить некоторые файлы из старой версии:

1. **Скопируйте изображения**
\`\`\`bash
# Скопируйте ваши изображения из старой версии
cp -r My-personal-portfolio-old/src/img portfolio-v2/src/
\`\`\`

2. **Обновите пути к изображениям**
В файле `Hero.jsx` замените пути:
\`\`\`jsx
// Вместо импорта несуществующих изображений
import person from '../../img/Person.png';

// Используйте свои изображения или удалите эти импорты
\`\`\`

3. **Кастомизируйте контент**
Откройте `Hero.jsx` и обновите:
- Проекты
- Навыки
- Текст About секции

---

### Вариант 3: Обновление на GitHub

1. **Создайте новую ветку**
\`\`\`bash
git checkout -b portfolio-v2
\`\`\`

2. **Удалите старые файлы (кроме .git)**
\`\`\`bash
rm -rf src public package.json
\`\`\`

3. **Скопируйте новые файлы**
\`\`\`bash
cp -r ../portfolio-v2/* .
\`\`\`

4. **Закоммитьте изменения**
\`\`\`bash
git add .
git commit -m "Portfolio V2: Complete redesign with 3D elements and modern animations"
git push origin portfolio-v2
\`\`\`

5. **Создайте Pull Request**
На GitHub создайте PR из ветки `portfolio-v2` в `main`

## ⚠️ Важные моменты

### 1. Изображения
Новая версия НЕ включает изображения из старой версии. Вам нужно:
- Скопировать их самостоятельно
- ИЛИ использовать новые изображения
- ИЛИ использовать placeholder'ы

### 2. Зависимости
Убедитесь что Node.js версии 16+:
\`\`\`bash
node --version
\`\`\`

### 3. Производительность
Новая версия оптимизирована, но Canvas анимации могут быть требовательны. 
Можно отключить в `ThreeBackground.jsx` если нужно.

## 🆘 Решение проблем

### Проблема: npm install не работает
**Решение:**
\`\`\`bash
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
\`\`\`

### Проблема: Ошибки при запуске
**Решение:**
\`\`\`bash
# Убедитесь что установлен react-scripts
npm install react-scripts --save
npm start
\`\`\`

### Проблема: Стили не применяются
**Решение:**
Убедитесь что установлен SCSS:
\`\`\`bash
npm install sass --save
\`\`\`

### Проблема: Анимации тормозят
**Решение:**
Уменьшите количество частиц в `ThreeBackground.jsx`:
\`\`\`js
// Было
const particleCount = Math.min(window.innerWidth / 10, 100);

// Станет
const particleCount = Math.min(window.innerWidth / 20, 50);
\`\`\`
